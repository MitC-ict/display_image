# display_image
複数画像をカード形式で表示

### 1. (初回のみ)　仮想環境作成
```
python -m venv venv
```

### 2. 仮想環境に入る
```
. venv/Scripts/activate
```

### 3. pipを最新にする
```
python -m pip install --upgrade pip
```

### 4. (初回のみ) モジュールをインストールする
```
pip install -r requirements.txt
```

### 5. 実行方法
```
streamlit run main.py
```
